# How to Contribute to This Project

## For Designers
1. Export designs from Figma as PNG (2x scale)
2. Upload to: `/designs/mockups/[screen-type]/`
3. Name files: `[screen-name]-v[number].png`
4. Create GitHub Issue when design is ready for development

## For Developers
1. Create a feature branch: `git checkout -b feature/your-feature`
2. Write code in `/code/` folder
3. Make small, meaningful commits
4. Push your branch: `git push origin feature/your-feature`
5. Open a Pull Request
6. Request review from lead developer
7. Once approved, merge to main

## Branch Naming
- Features: `feature/login-screen`
- Bugs: `fix/button-padding-issue`
- Chores: `chore/update-docs`

## Commit Message Format
- `feat: add new feature`
- `fix: fix bug`
- `docs: update documentation`
- `chore: routine maintenance`
