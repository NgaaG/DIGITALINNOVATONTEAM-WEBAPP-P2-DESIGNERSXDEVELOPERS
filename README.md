## Getting Started

### For Designers
1. Export Figma screens as PNG (2x scale)
2. Upload to `/designs/mockups/[screen-type]/`
3. Create GitHub Issue when ready for development

### For Developers
1. Clone this repo
2. Create a feature branch
3. Build from Figma mockups
4. Open Pull Request when ready
5. Request review from lead developer

## Tech Stack
